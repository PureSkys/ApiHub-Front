# 学校管理功能开发计划

## 一、需求概述

在前端系统中添加一个全新的一级菜单"学校管理"，该菜单下包含以下子菜单：
- **概览** - 学校管理系统的数据统计概览
- **用户管理** - 学校管理员账户管理
- **学校管理** - 学校信息的增删改查
- **班级管理** - 班级信息的增删改查
- **学生管理** - 学生信息的增删改查
- **考试管理** - 考试信息的增删改查
- **成绩管理** - 学生成绩的录入与管理

## 二、技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus + TailwindCSS
- **状态管理**: Pinia
- **路由**: Vue Router (Hash模式)
- **HTTP请求**: Axios

```typescript
{

  children: [

    { path: 'overview', name: 'SchoolOverview', component: () => import('@/views/school/SchoolOverview.vue'), meta: { title: '概览', icon: 'DataAnalysis', requiresAuth: true } },

    { path: 'classes', name: 'SchoolClasses', component: () => import('@/views/school/SchoolClasses.vue'), meta: { title: '班级管理', icon: 'Grid', requiresAuth: true } },

```

### 4.2 API模块开发
**目录**: `src/api/school.ts`

创建学校管理相关的API接口封装，包含：
- TypeScript类型定义（请求/响应类型）
- 所有学校管理相关API函数

### 4.3 页面组件开发
**目录**: `src/views/school/`

创建以下页面组件：

| 组件 | 文件名 | 功能描述 |
|------|--------|----------|
| 概览页 | SchoolOverview.vue | 展示学校统计数据、图表 |
| 用户管理 | SchoolUsers.vue | 学校管理员的增删改查、启用/禁用 |
| 学校管理 | Schools.vue | 学校信息的增删改查 |
| 班级管理 | SchoolClasses.vue | 班级信息的增删改查，关联学校 |
| 学生管理 | SchoolStudents.vue | 学生信息的增删改查，关联班级 |
| 考试管理 | SchoolExams.vue | 考试信息的增删改查，关联学校 |
| 成绩管理 | SchoolScores.vue | 成绩录入、查询、统计 |

### 4.4 代码风格遵循

参照现有代码风格：
- 使用 Vue 3 Composition API (`<script setup lang="ts">`)
- 使用 Element Plus 组件库
- 使用 TailwindCSS 进行样式
- 表格使用 `el-table`，表单使用 `el-form`
- 对话框使用 `el-dialog`
- 消息提示使用 `ElMessage`、`ElMessageBox`
- 响应式设计支持移动端

## 七、注意事项

1. **不修改现有功能**: 仅添加新菜单和相关功能，不修改已有的句子集等功能
2. **保持代码风格一致**: 参照现有代码风格进行开发
3. **响应式设计**: 所有页面需要支持移动端显示
4. **错误处理**: 统一使用 `ElMessage` 进行错误提示
5. **权限控制**: 所有页面都需要登录认证
