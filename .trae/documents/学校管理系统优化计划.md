# 学校管理系统优化计划

## 一、任务概述

1. 修复学生管理模块中"查看学生成绩"功能的故障
2. 重构成绩管理模块的操作流程（学校→考试→成绩）
3. 重新开发统计分析功能

## 二、详细实施计划

### 任务1：修复学生管理"查看学生成绩"功能

**当前问题分析**：
- 文件：`src/views/school/SchoolStudents.vue`
- 当前成绩展示弹窗只显示部分科目（语文、数学、英语、物理、化学、生物）
- 缺少：历史、政治、地理及赋分成绩
- API：`/school/student/{student_id}/scores` 返回 `ScoreResponse[]`

**修复方案**：
1. 扩展成绩展示表格，添加所有科目列：
   - 文科科目：历史、政治、地理
   - 赋分科目：化学赋分、生物赋分、政治赋分、地理赋分
2. 添加总分计算和展示
3. 添加成绩统计信息（平均分、排名等）

### 任务2：重构成绩管理模块操作流程

**当前问题分析**：
- 文件：`src/views/school/SchoolScores.vue`
- 当前筛选：直接选择考试和学生
- 问题：没有学校层级，无法按学校筛选考试

**重构方案**：
1. 修改筛选流程为：学校 → 考试 → 成绩列表
2. 添加学校选择器作为第一级筛选
3. 考试选择器根据所选学校动态加载
4. 成绩列表根据所选考试展示

**具体修改**：
```
筛选区域布局：
[选择学校] → [选择考试] → [筛选学生（可选）]
```

### 任务3：重新开发统计分析功能

**当前问题分析**：
- 文件：`src/views/school/SchoolStats.vue`
- 当前已实现部分统计功能
- 需要验证API返回数据结构并优化展示

**已有统计API**：
1. `getSchoolOverviewStats` - 学校概览统计
2. `getClassDetailStats` - 班级详细统计
3. `getExamOverviewStats` - 考试概览统计
4. `getScoreDistribution` - 成绩分布
5. `getExamRanking` - 考试排名
6. `getClassSubjectComparison` - 班级科目对比
7. `getStudentExamComparison` - 学生考试对比
8. `getExamClassComparison` - 考试班级对比
9. `getExamPassRate` - 考试及格率

**优化方案**：
1. 考试分析模块：
   - 保持现有功能，验证数据展示
   - 优化图表配置

2. 学生成绩趋势模块：
   - 保持现有功能
   - 添加更多统计维度

3. 班级科目分析模块：
   - 保持现有功能
   - 优化数据展示

## 三、实施步骤

### 步骤1：修复学生成绩查看功能
1. 修改 `SchoolStudents.vue` 中的成绩展示弹窗
2. 添加所有科目列（历史、政治、地理及赋分）
3. 添加总分列
4. 添加成绩统计信息

### 步骤2：重构成绩管理筛选流程
1. 在 `SchoolScores.vue` 添加学校选择器
2. 修改考试选择器为动态加载
3. 调整筛选逻辑

### 步骤3：优化统计分析功能
1. 验证各统计API返回数据
2. 优化图表展示
3. 添加数据为空时的提示

## 四、预期成果

1. 学生成绩查看功能完整展示所有科目成绩
2. 成绩管理模块实现学校→考试的层级筛选
3. 统计分析功能正常工作，图表展示直观清晰
